First rule of the position property - Don't abuse the position property!

CSS Position

The Position Property can have 5 values:

  1. Static
  2. Relative
  3. Fixed
  4. Absolute
  5. Sticky

Once we put a position on an element, we can adjust its position with the top, right, bottom, and left properties.

Position - Static

The default value of an element's position property.

Position - Relative

The element is positioned relatively to its normal position.

Position - Fixed

The element is positioned relatively to the viewport.

Position - Absolute

The element is positioned relatively to its nearest positioned ancestor. If there is no positioned ancestor, the element is positioned relative to the document body.

An element with position:absolute should always have a parent with position: relative.

Position - Sticky

The element is positioned relative to the user's scroll position. The navbar on this page is positioned using position:sticky.

Sticky toggles between relative and fixed.
